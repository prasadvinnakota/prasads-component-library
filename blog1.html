Lets Learn MAP, FILTER, REDUCE

Map

Map takes the each element of the array and applies the condition to every element inside it. It returns an array of elements on applying the condition.

const arr1=[1,2,3]

arr1.map((index,num)=>console.log(num*index)) //This will console all numbers in it by multiplying with index. Here index is optional.

// o/p: 0 2 6

Filter

Filter takes an array of elements and filters the array based on the condition. It return a new array with filtered elements(where our condition is true).

const arr1=[1,2,3]

const arr2 = arr1.filter((index,num)=>num%index===0) //This will give a array where the numbers are divided by index. Here index is optional.

// o/p: [1,2]

Reduce

Reduce takes an array and gives out single value. It goes through each and every element of array and gives the output based on the condition we give.

const arr1=[1,2,3]

const sum= arr1.reduce((total,current)=>total+current,0)

console.log(sum) // o/p:6
